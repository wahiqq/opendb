# RISE Research Brand Guidelines

**IMPORTANT**: Always follow these guidelines for all design and development work.

## Color Palette

**DO NOT USE BROWN (#83684f)** - This color is NOT part of the brand.

### Official Colors:
- **Primary Green**: `#1C7C54` - Use for primary actions, highlights, and brand elements
- **Beige/Cream**: `#FBF8EF` - Use for backgrounds
- **White**: `#FFFFFF` - Use for cards and content areas
- **Black**: `#000000` - Use for text and icons

## Typography

**Font Family**: `Red Hat Display`
- Already configured in the CSS
- Use for all text throughout the application

## Logo

Location: `/static/rise-logo.png`

The logo features:
- A tree icon on the left
- "RISE RESEARCH" text on the right
- Black color scheme

## Design Principles

1. Use green (#1C7C54) for all primary actions and brand highlights
2. Use beige (#FBF8EF) for page backgrounds
3. Use white for cards and content containers
4. Use black for text
5. **Never use brown colors**
6. Always use Red Hat Display font
7. Keep the design clean and minimal
